# Student Dashboard Requirements Document

## 1. Dashboard

- Show Cards each one clickable and redirect to the respective page
    - Upcoming Quizzes
    - Classes

## Classes Page

- Show all classes sorted alphabetically with a search function
- Show classes as cards each card has the class name, number of students, and teacher name

## Quiz Management Page

- Show all quizzes sorted by date desc
- Show Quizzes as cards each card has the quiz name, description and start and end dates, and status(open, ended, upcoming), teacher name
- Each card has 3 buttons
    - Show Details
    - View Submission(if score is released)
    - Attempt Quiz(if quiz is open)

## Submission Details Page

- Show the Student's Answers
- Show the Correct Answers
- Show the Score
- Show the Time Taken
- Show the Date and Time of Submission
